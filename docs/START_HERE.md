## 📋 What Happens When You Deploy

1. Your code goes to GitHub repository: https://github.com/jdgraves1995/lovebankapp
2. GitHub Pages reads your code and hosts it
3. Your live app becomes available at the GitHub Pages URL
4. The app loads index.html and runs in users' browsers
5. Data is stored locally in each user's browser
