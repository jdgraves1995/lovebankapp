## Next Steps: Upload to GitHub

### 1. Create GitHub Repository
- Visit https://github.com/new
- Create repo named `love-bank` (or your preference)
- Keep it public
- **Do NOT add README, gitignore, or license** (already included)

### 2. Connect Local Repo to GitHub
Open PowerShell and run these commands (replace YOUR_USERNAME):

```powershell
cd "c:\VisualStudioCodeWorkspaces\VisualStudioCodePromptEngineer\love-bank-app"
git remote add origin https://github.com/YOUR_USERNAME/love-bank.git
git branch -M main
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to your GitHub repo
2. Click Settings → Pages
3. Set source to "Deploy from a branch"
4. Select branch: `main`, folder: `/root`
5. Save

### 4. Access Your Live Site
Your app will be live at:
```
https://YOUR_USERNAME.github.io/love-bank
```
